# Web Veterinaria Tarapacá

Sitio web oficial de la Clínica Veterinaria Tarapacá en Iquique, Chile.

## 🏥 Descripción

Sitio web moderno y responsivo para la Clínica Veterinaria Tarapacá, especializada en consultas generales y endocrinología veterinaria. El sitio incluye:

- **Sistema de agendamiento online** integrado con Google Calendar
- **Presentación del equipo profesional** liderado por la Dra. Javiera Solis Larenas  
- **Servicios veterinarios** completos y especializados
- **Integración con redes sociales** (TikTok e Instagram)
- **Diseño responsivo** optimizado para móviles y desktop
- **SEO local** optimizado para búsquedas en Iquique

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Framework CSS**: Tailwind CSS
- **Fuentes**: Poppins (Google Fonts)  
- **Iconos**: Font Awesome
- **Backend**: Google Apps Script
- **Calendar API**: Google Calendar Integration
- **Deployment**: Vercel
- **Redes Sociales**: TikTok y Instagram Embeds

## 🚀 Características del Sistema

### Sistema de Agendamiento
- Calendario interactivo con disponibilidad en tiempo real
- Integración directa con Google Calendar
- Formulario de datos completo (tutor y mascota)
- Confirmación automática por email
- Horarios flexibles según tipo de consulta

### Diseño y UX
- Interfaz estilo RedSalud para booking médico
- Paleta de colores corporativa (naranja #ff9500, marrón #8B4513)
- Animaciones suaves y transiciones optimizadas
- Compatible con Safari, Chrome, Firefox y Edge

### SEO y Performance  
- Meta tags optimizados para búsquedas locales
- Schema markup para negocios locales
- Core Web Vitals optimizados
- Lazy loading de imágenes
- Compresión de assets

## 🔧 Instalación y Desarrollo

```bash
# Clonar repositorio
git clone https://github.com/veterinariatarapaca-lab/web-veterinaria-tarapaca.git

# Navegar al directorio
cd web-veterinaria-tarapaca

# Servir localmente (Python)
python -m http.server 8000

# O con Node.js
npx serve . -p 8000

# Abrir en navegador
http://localhost:8000
```
